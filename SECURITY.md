# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| < 0.4   | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities privately through GitHub Security Advisories:

https://github.com/ansoragroup/geekshop-ui/security/advisories/new

**Do not open a public issue for security vulnerabilities.**

## Response Timeline

- **Acknowledge:** within 72 hours of report.
- **Fix (critical):** within 30 days.
- **Fix (moderate/low):** within 90 days.

## Coordinated Disclosure

Public disclosure is coordinated after a fix is released. We will credit reporters unless they prefer to remain anonymous.
