# Spacing & Border Radius

Consistent spacing and radius tokens ensure visual harmony across all components.

---

<SectionTitle>Spacing Scale</SectionTitle>

The spacing scale uses a progression from 2px to 32px. All spacing values are available as SCSS variables with the `$spacing-*` prefix.

### Usage Guidelines

| Context | Token | Value |
|---|---|---|
| Tight inline spacing (icon + text) | `$spacing-xs` | 4px |
| Default gap between elements | `$spacing-sm` | 8px |
| Internal card padding | `$spacing-lg` | 16px |
| Container horizontal padding | `$spacing-md` | 12px |
| Section vertical spacing | `$spacing-xxl` | 24px |
| Page-level section gaps | `$spacing-xxxl` | 32px |
| Minimum touch target spacing | `$spacing-sm` | 8px |

### SCSS Usage

```scss
@use '../../../theme/tokens' as *;

.card {
  padding: $spacing-lg;
  gap: $spacing-md;
}

.section {
  margin-bottom: $spacing-xxl;
}

.inline {
  gap: $spacing-xs;
}
```

### Usage Guidelines

| Element | Token | Value |
|---|---|---|
| Tags, small badges | `$radius-xs` | 4px |
| Buttons, inputs | `$radius-sm` | 6px |
| Cards, dropdowns | `$radius-md` | 8px |
| Large cards, modals | `$radius-lg` | 12px |
| Bottom sheets, overlays | `$radius-xl` | 16px |
| Large banners | `$radius-xxl` | 24px |
| Pills, circular avatars | `$radius-round` | 9999px |

### SCSS Usage

```scss
@use '../../../theme/tokens' as *;

.button {
  border-radius: $radius-sm;
}

.card {
  border-radius: $radius-lg;
}

.pill {
  border-radius: $radius-round;
}
```

<SectionTitle>Layout Constants</SectionTitle>

Fixed dimensions used for structural elements:

| Token | Value | Purpose |
|---|---|---|
| `$navbar-height` | 44px | Top navigation bar |
| `$tabbar-height` | 50px | Bottom tab bar |
| `$action-bar-height` | 56px | Bottom action bar (cart, buy) |
| `$search-bar-height` | 36px | Search input bar |
| `$container-padding` | 12px | Page container horizontal padding |
| `$safe-area-bottom` | `env(safe-area-inset-bottom, 0px)` | iOS safe area |

These are also available as CSS custom properties:

```css
var(--gs-navbar-height)    /* 44px */
var(--gs-tabbar-height)    /* 50px */
var(--gs-action-bar-height) /* 56px */
var(--gs-safe-area-bottom)  /* env() */
```
