# @ansoragroup/ui

> E-commerce component library for React, built mobile-first with container query adaptivity.

## Key Features

- **55+ Components** across 8 categories -- Commerce, Content, Data Display, Feedback, Form, Layout, Navigation, Product
- **4 Hooks** -- useControllableState, useCountdown, useFocusTrap, useToast
- **Mobile-first** -- Designed for 390px viewport with container query adaptivity
- **Accessible** -- WCAG 2.1 AA compliant, keyboard navigable
- **Themeable** -- CSS Custom Properties (`--gs-*` prefix), dark mode support
- **Tree-shakeable** -- ESM + CJS output with preserveModules
- **TypeScript** -- Full type safety with exported Props interfaces
- **i18n Ready** -- Built-in localization for Uzbek, Russian, English
