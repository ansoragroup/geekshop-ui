# Typography

The design system uses the system font stack for optimal performance and native feel across platforms.

---

<SectionTitle>Font Family</SectionTitle>

```
$font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
              'Helvetica Neue', Arial, sans-serif;
```

The system font stack renders the platform's native typeface -- San Francisco on Apple devices, Segoe UI on Windows, Roboto on Android.

<SectionTitle>Usage Guidelines</SectionTitle>

| Element | Size | Weight | Line Height |
|---|---|---|---|
| Page title | `xxxl` (24px) | Bold (700) | Tight (1.2) |
| Section header | `xxl` (20px) | Semibold (600) | Tight (1.2) |
| Card title | `lg` (16px) | Semibold (600) | Normal (1.5) |
| Body text | `md` (14px) | Regular (400) | Normal (1.5) |
| Secondary text | `sm` (12px) | Regular (400) | Normal (1.5) |
| Caption / label | `xs` (10px) | Regular (400) | Normal (1.5) |
| Primary price | `price-xl` (30px) | Bold (700) | Tight (1.2) |
| Card price | `price-md` (18px) | Bold (700) | Tight (1.2) |
| Inline price | `price-sm` (14px) | Bold (700) | Tight (1.2) |
| Button text | `md` (14px) | Semibold (600) | Tight (1.2) |
| Tab / nav label | `sm` (12px) | Medium (500) | Tight (1.2) |
| Badge text | `xs` (10px) | Medium (500) | Tight (1.2) |

### SCSS Token Usage

```scss
@use '../../../theme/tokens' as *;

.title {
  font-size: $font-size-lg;
  font-weight: $font-weight-semibold;
  line-height: $line-height-tight;
}

.body {
  font-size: $font-size-md;
  font-weight: $font-weight-regular;
  line-height: $line-height-normal;
}

.caption {
  font-size: $font-size-xs;
  color: var(--gs-text-tertiary);
}
```
