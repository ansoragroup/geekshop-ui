# Icons

`@ansoragroup/ui` uses [lucide-react](https://lucide.dev/) as its icon library. Lucide provides a consistent set of 1000+ icons with a unified stroke style that matches the library's visual language.

---

<SectionTitle>Installation</SectionTitle>

`lucide-react` is a peer dependency. Install it alongside the library:

```bash
npm install lucide-react
```

<SectionTitle>Basic Usage</SectionTitle>

Import individual icons by name:

```tsx
import { ShoppingCart, Heart, Star } from 'lucide-react'

function MyComponent() {
  return (
    <div style={{ display: 'flex', gap: 8 }}>
      <ShoppingCart size={20} />
      <Heart size={20} />
      <Star size={20} fill="currentColor" />
    </div>
  )
}
```

---

<SectionTitle>Size Convention</SectionTitle>

Use these sizes consistently across your application:

<SizeDemo size={16} label="Inline — inside text, labels, badges" />
<SizeDemo size={20} label="Buttons — inside Button, ActionBar, NavBar" />
<SizeDemo size={24} label="Standalone — section headers, empty states" />

```tsx
// Inline (12-16px) — inside text or small badges
<Tag><Star size={12} /> Featured</Tag>

// Button (20px) — action buttons, navigation
<Button icon={<ShoppingCart size={20} />}>Add to Cart</Button>

// Standalone (24px) — empty states, section icons
<Empty icon={<Package size={24} />} />
```

---

<SectionTitle>Color Convention</SectionTitle>

Icons inherit color from their parent element by default via `stroke="currentColor"`. This ensures they match text color automatically.

```tsx
// Inherits parent text color (recommended)
<span style={{ color: '#FF5000' }}>
  <Heart size={20} /> {/* renders orange */}
</span>

// Explicit color override
<Star size={20} stroke="#FFA726" fill="#FFA726" />

// Using CSS custom properties
<ShoppingCart size={20} style={{ color: 'var(--gs-color-primary)' }} />
```

**Filled icons** — Some icons like Star and Heart are commonly shown filled. Use the `fill` prop:

```tsx
<Star size={16} fill="currentColor" />          {/* filled star */}
<Heart size={16} fill="currentColor" />         {/* filled heart */}
<Star size={16} fill="none" />                  {/* outline star (default) */}
```

<SectionTitle>Usage in Components</SectionTitle>

Many library components accept icon props or use icons internally:

### ActionBar

```tsx
import { ActionBar } from '@ansoragroup/ui'
import { ShoppingCart, Heart, MessageCircle } from 'lucide-react'

<ActionBar
  onCartClick={() => {}}
  onBuyClick={() => {}}
  onFavoriteClick={() => {}}
  onContactClick={() => {}}
/>
```

The ActionBar, NavBar, and AppBar components use lucide-react icons internally (ShoppingCart, Heart, ArrowLeft, Share2, etc.). You do not need to pass icons directly -- they are built in.

### Button with icon

```tsx
import { Button } from '@ansoragroup/ui'
import { ShoppingCart } from 'lucide-react'

<Button
  variant="primary"
  size="md"
  icon={<ShoppingCart size={20} />}
>
  Add to Cart
</Button>
```

```tsx
import { SearchBar } from '@ansoragroup/ui'
import { Camera } from 'lucide-react'

<SearchBar
  placeholder="Search products..."
  onCameraClick={() => {}}
/>
```

<SectionTitle>Custom Icons</SectionTitle>

If you need an icon not available in lucide-react, create a custom SVG component that matches the library's icon style:

```tsx
// Match lucide's defaults: 24x24 viewBox, 1.5px stroke
function CustomIcon({ size = 24, ...props }) {
  return (
    <svg
      xmlns="http://www.w3.org/2000/svg"
      width={size}
      height={size}
      viewBox="0 0 24 24"
      fill="none"
      stroke="currentColor"
      strokeWidth="1.5"
      strokeLinecap="round"
      strokeLinejoin="round"
      {...props}
    >
      {/* your paths here */}
    </svg>
  )
}
```

**Style rules for custom icons:**
- **viewBox:** `0 0 24 24`
- **stroke-width:** `1.5` (matches lucide default)
- **stroke-linecap:** `round`
- **stroke-linejoin:** `round`
- **fill:** `none` (outline style; use `currentColor` only for intentionally filled icons)
- **color:** inherit via `stroke="currentColor"`
