## Instructions

This is a lightweight version of consilium for tasks that need multi-perspective thinking
but not the full 9-phase protocol.

1. Spawn 3 agents based on task analysis:
   - Always: one generalist/architect
   - Two domain specialists based on what the task touches

2. Each agent: investigate their domain (2 min max), write a brief proposal

3. One round of cross-review: each reads others' proposals, writes objections

4. Lead synthesizes into a single recommendation

5. Execute (or present for approval if Level 1)

6. Brief retrospective: update metrics.jsonl, knowledge-graph.jsonl, owner-profile.jsonl

Skip Phases 2 (cross-pollinate) and 6 (formal review). Keep it fast.

Load memory/ for past learnings. Update memory/ when done.
